# Descripción

Este proyecto implementa una lista doblemente enlazada en Python con:
- Inserción al principio
- Inserción al final
- Eliminación por valor (carnet)
- Visualización en consola
- Representación gráfica con Graphviz

## Cómo ejecutar.

1. Instalar dependencias:
   pip install graphviz

2. Ejecutar:
   python lista_doble.py

Cada modificación genera automáticamente una imagen llamada:
lista_doble.png

## Requisitos.

- Python 3.x
- Graphviz instalado en el sistema
